# Redink Examples

This directory contains practical CLI usage examples.

Each subdirectory demonstrates a specific usage scenario:

- `basic_scan/` → Default scan
- `advanced_scan/` → Customized and high-control scan
- `risk_only_mode/` → Risk-focused reporting

These examples serve as functional documentation and CLI contract references.

Each example includes:

- Input data
- Command used
- Expected output
- Context explanation (when needed)

These examples are intended for:

- Manual testing
- Demonstrations
- CI validation
- Academic documentation
