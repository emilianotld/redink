# Code of Conduct

## Purpose

Redink is a professional, security-oriented project focused on clarity, ethics, and responsible engineering.

This Code of Conduct exists to ensure a respectful, collaborative, and constructive environment for everyone involved.

---

## Expected Behavior

All participants are expected to:

- Communicate respectfully and professionally
- Accept constructive criticism gracefully
- Focus discussions on technical merit and project goals
- Act in good faith when contributing or reporting issues

---

## Unacceptable Behavior

The following behaviors will not be tolerated:

- Harassment, discrimination, or personal attacks
- Aggressive, hostile, or dismissive communication
- Promotion of unethical, illegal, or intrusive use of the tool
- Repeated disregard for project guidelines or scope

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- Pull requests
- Discussions and reviews
- Any communication related to Redink

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Violations may result in warnings, removal of content, or exclusion from the project, depending on severity.

---

## Contact

If you experience or witness unacceptable behavior, please contact the project maintainer directly.
